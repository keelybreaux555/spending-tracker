# Spending Tracker (Python)

## Description
A simple Python program that allows users to input expenses and calculates the total amount spent.

## Features
- Add multiple expenses
- Categorize spending (food, gas, etc.)
- Calculates total spending

## Example Output
Total spent: 85.0

## What I Learned
- How to take user input in Python
- How to store data in lists
- How to loop through data
- How to calculate totals

## How to Run
1. Open terminal
2. Run: python spending_tracker.py
